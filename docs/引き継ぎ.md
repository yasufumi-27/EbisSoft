## 1. 新セッションが最初に知るべきこと（重要・つまずき防止）

1. **これは“いつものNext.js”ではない**：**Next.js 16**（App Router / Turbopack）。APIが学習データと異なる場合がある。
   コードを書く前に `node_modules/next/dist/docs/` の該当ガイドを読む（`AGENTS.md` の指示）。
2. **Bashサンドボックスがネットワークを遮断している**：`npm install` / `npm run build`（`next/font`のフォント取得）/ 外部取得は
   **`dangerouslyDisableSandbox: true` を付けて実行**する。付けないと `ENOTFOUND registry.npmjs.org` で失敗する。
3. **シェルは zsh**：`shopt` 等のbash builtinは使えない。必要なら `bash -c '...'` で実行する。
4. **単一情報源（ここだけ直せば全体に反映）**：
   - `src/lib/site.ts` … 社名・URL・NAP・SNS・keywords・knowsAbout
   - `src/lib/content.ts` … 掲載内容（サービス/料金/実績/お客様の声/FAQ/要点/AEO）
   - `src/lib/jsonld.ts` … 構造化データ（JSON-LD）ビルダー
   - `src/app/globals.css` … ブランドカラー・フォント（`@theme`）
5. **`★` 印 = 差し替え必須のプレースホルダ**（`site.ts` 等）。
6. **検証コマンド**（下記5章）でビルド・型・Lint・実機HTMLを必ず確認する。
7. **技術メモ**：Tailwind **v4**（`@theme`／CSS変数）。`"use server"` ファイルは**async関数しかexport不可**
   （型・定数は別ファイルへ。例：`src/app/contact-state.ts`）。OG画像のsatori既定フォントは**日本語非対応**（画像内はラテン文字主体）。
